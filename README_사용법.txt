=============================================
  Stock Predictor - 사용법
=============================================

[프로그램 설명]
NASDAQ/AMEX 종목을 기술적 분석, 모멘텀, 펀더멘털, 머신러닝(XGBoost+RandomForest)
4가지 방법으로 종합 분석하여 단기 상승 확률이 높은 종목을 추천합니다.

[사전 요구사항]
- Python 3.9 이상 (https://www.python.org/downloads/)
  설치 시 "Add Python to PATH" 반드시 체크!

[실행 방법]
1. Windows:
   - start.bat 더블클릭

2. WSL/Linux:
   - chmod +x start.sh
   - ./start.sh

3. 브라우저에서 http://localhost:8000 접속

[안드로이드에서 사용 (PWA)]
1. PC와 안드로이드가 같은 Wi-Fi에 연결되어 있어야 함
2. PC의 IP 주소 확인 (cmd에서 ipconfig)
3. 안드로이드 Chrome에서 http://PC_IP:8000 접속
4. Chrome 메뉴(⋮) → "홈 화면에 추가" 클릭
5. 앱처럼 아이콘이 생기고, 전체화면으로 사용 가능!

[Windows 방화벽 설정]
안드로이드에서 접속이 안 되면:
1. Windows 설정 → Windows 보안 → 방화벽 및 네트워크 보호
2. "고급 설정" → "인바운드 규칙" → "새 규칙"
3. 포트 → TCP → 8000 → 허용

[분석 방법 설명]
1. 기술적 분석 (25%): RSI, MACD, 볼린저밴드, 스토캐스틱, ADX, CCI, MFI 등
2. 모멘텀 분석 (25%): 가격 모멘텀, 거래량 급증, 돌파 패턴, 갭 분석, 변동성 수축
3. 펀더멘털 분석 (20%): P/E, PEG, PBR, 매출/이익 성장률, ROE, 부채비율
4. 머신러닝 (30%): XGBoost + RandomForest 앙상블 (50+ 피처)

[등급 체계]
S등급 (85+): 강력 매수 추천
A등급 (70-84): 매수 추천
B등급 (55-69): 관심 종목
C등급 (40-54): 중립
D등급 (25-39): 보류
F등급 (0-24): 비추천

[주의사항]
- 이 프로그램은 참고용이며, 실제 투자 결정의 책임은 본인에게 있습니다.
- 주식 시장은 예측 불가능한 요소가 많으므로 분산 투자를 권장합니다.
- 무료 Yahoo Finance API를 사용하므로 데이터 지연이 있을 수 있습니다.
- 분석에 약 3~10분 소요됩니다 (종목 수, 인터넷 속도에 따라 다름).
